# 🛠️ Python Toolkit

A comprehensive collection of useful tools built with Streamlit for media processing, file management, and productivity.

## 🚀 Features

### 🎬 Media Downloader
- Download videos and audio from YouTube, Instagram, TikTok, X (Twitter), and LinkedIn
- Smart defaults with one-click MP3 downloads
- Customizable quality and format options
- Persistent download progress tracking
- Modal-based configuration for clean UX

### 🎙️ Podcast Summarizer
- Extract audio from podcast URLs
- AI-powered transcription and summarization
- Customizable summary styles and focus areas
- Support for timestamps and action items
- Download summaries and transcripts

## 📋 Requirements

- Python 3.8+
- Streamlit
- yt-dlp
- Other dependencies listed in `requirements.txt`

## 🔧 Installation

1. Clone the repository:
```bash
git clone https://github.com/PatrickWiloak/toolkit-python.git
cd toolkit-python
```

2. Install dependencies:
```bash
pip install -r requirements.txt
```

3. Run the application:
```bash
streamlit run toolkit.py
```

Or use the provided start script:
```bash
./start
```

## 🎯 Usage

1. **Launch the toolkit**: Run the start script or use Streamlit directly
2. **Select a tool**: Choose from the available tools on the home page
3. **Configure options**: Use the clean, modal-based interfaces
4. **Process content**: Monitor progress with persistent status displays

## 🛡️ Features

- **Clean UX**: Simplified interfaces with smart defaults
- **Persistent Progress**: Download/processing status that doesn't disappear
- **Navigation Protection**: Prevents accidental navigation during operations
- **Modal Configuration**: Non-blocking setup flows
- **Smart Defaults**: One-click operations with sensible presets
